# receiveCompleteReturnValidationStep - Medusa Core Workflows Reference

This documentation provides a reference to the `receiveCompleteReturnValidationStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that a return can be received and completed.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { receiveCompleteReturnValidationStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = receiveCompleteReturnValidationStep({
      "orderReturn": "{value}",
      "input": {
        "return_id": "{value}",
        "items": [{
          "id": "id_inzn4BfRYBpJJzN2",
          "quantity": 1
        }]
      }
    })
  }
)
```
