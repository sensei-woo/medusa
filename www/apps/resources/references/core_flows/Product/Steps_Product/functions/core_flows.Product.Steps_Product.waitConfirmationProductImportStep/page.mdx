# waitConfirmationProductImportStep - Medusa Core Workflows Reference

This documentation provides a reference to the `waitConfirmationProductImportStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step waits until a product import is confirmed.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { waitConfirmationProductImportStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = waitConfirmationProductImportStep()
  }
)
```
