# createVariantPricingLinkStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createVariantPricingLinkStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates links between variant and price set records.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { createVariantPricingLinkStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = createVariantPricingLinkStep({
      "links": [{
        "variant_id": "{value}",
        "price_set_id": "{value}"
      }]
    })
  }
)
```
