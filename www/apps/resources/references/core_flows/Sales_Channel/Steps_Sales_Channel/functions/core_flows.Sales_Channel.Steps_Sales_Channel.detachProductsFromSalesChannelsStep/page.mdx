# detachProductsFromSalesChannelsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `detachProductsFromSalesChannelsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step dismisses links between product and sales channel records.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { detachProductsFromSalesChannelsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = detachProductsFromSalesChannelsStep({
      "links": [{
        "sales_channel_id": "{value}",
        "product_id": "{value}"
      }]
    })
  }
)
```
