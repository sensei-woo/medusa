# associateProductsWithSalesChannelsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `associateProductsWithSalesChannelsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates links between products and sales channel records.

import { associateProductsWithSalesChannelsStep } from "@medusajs/medusa/core-flows"
