# detachLocationsFromSalesChannelsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `detachLocationsFromSalesChannelsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step dismisses links between location and sales channel records.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { detachLocationsFromSalesChannelsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = detachLocationsFromSalesChannelsStep({
      "links": [{
        "sales_channel_id": "{value}",
        "location_id": "{value}"
      }]
    })
  }
)
```
