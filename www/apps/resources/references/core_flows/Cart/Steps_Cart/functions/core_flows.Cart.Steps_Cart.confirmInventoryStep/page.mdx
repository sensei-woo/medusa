# confirmInventoryStep - Medusa Core Workflows Reference

This documentation provides a reference to the `confirmInventoryStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step confirms for one or more variants that their inventory has a required quantity.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { confirmInventoryStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = confirmInventoryStep({
      "items": [{
        "inventory_item_id": "{value}",
        "required_quantity": 48,
        "allow_backorder": true,
        "quantity": 33,
        "location_ids": [
          "{value}"
        ]
      }]
    })
  }
)
```
